Sizing rules (percentage-based — library requirement)
Property Value Approx px at 1400px viewport
defaultSize 15% ~210px (matches current 200px sidebar)
minSize 11% ~154px — floor before snap-to-collapse
maxSize 28% ~390px — ceiling
collapsible true Drag below minSize → snaps to 0 (hidden)
autoSaveId "tendriv-shell" localStorage key — persists width + collapse state
Drag handle styling
The PanelResizeHandle renders a 4px wide bar. We style it with the jade token (--jade) on [data-resize-handle-active] and :hover, and a neutral --border color at rest. Three dot grips are centered vertically. Smooth 150ms colour transition.
Toggle button
Absolutely positioned on the left edge of the main panel. Jade pill, 14×28px, border-radius: 0 6px 6px 0. Uses panelRef.collapse() / panelRef.expand() from the library. Chevron flips (‹ / ›) based on panel.isCollapsed(). Button stays visible at all times so the user can always re-open the sidebar.
